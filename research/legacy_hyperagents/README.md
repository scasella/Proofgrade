# Legacy HyperAgents Archive

This folder keeps the older multi-domain and self-improvement scaffolding that was part of the original research-oriented repository shape.

It is retained for provenance only.

It is **not** part of the supported `proofgrade` runtime, **not** part of the public product surface, and **not** part of the public v0.1.0 benchmark claim.

If you are trying to:

- grade proofs locally
- run the public API or CLI
- reproduce the frozen `imo_grading` result package

you do **not** need anything in this archive.

The supported public surfaces live in:

- `proofgrade/`
- `configs/runtime/`
- `configs/baseline_freeze/`
- `analysis/` for the final frozen IMO result scripts
- `docs/`
- `artifacts/release/v0.1.0/`
