# v0.1.0 Release Artifacts

This directory is the curated public result bundle for the frozen `imo_grading` release line.

Intended contents:

- `ablation_summary.json`
- `lockbox_summary.json`
- `fresh_generalization_summary.json`
- `final_remaining_error_summary.json`
- `main_result_table.md`
- `mechanism_ablation_table.md`
- `error_bucket_table.md`
- `casebook.json`
- `casebook.md`

These files are curated public summaries copied from the frozen evaluation line after local verification.

They are meant to be:

- small enough to inspect quickly
- sufficient to rebuild the public tables and casebook
- narrower than the internal analysis outputs

The detailed frozen run data stays outside this public bundle. This directory is the public-facing release package, not a dump of every intermediate artifact.
